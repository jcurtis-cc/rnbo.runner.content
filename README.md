# rnbo.runner.content

This repo has some patches that we've built for the [rnbo.oscquery.runner](https://github.com/Cycling74/rnbo.oscquery.runner) aka "rnbo runner".
Initially we're targeting [RNBO Move Takeover](https://cycling74.com/products/rnbo/move) but the patchers are likely good to run on the normal pi without issue.
